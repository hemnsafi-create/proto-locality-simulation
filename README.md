- `order_parameters.py` — χ*_HHI, Q*, λ₂

- `scaling_sweep.py` — Table 4: N scaling
- `parameter_sweep.py` — Table 2: parameter sensitivity
- `entropy_analysis.py` — Section 4.5: relational entropy
- `visualize.py` — All paper figures
